üî∑ 1. What are Reference Data Types?

These are types whose variables store references (addresses) of objects in heap memory, not the actual value (unlike primitives which are stored in stack).

üì¶ Examples:

  • Class
  • Interface
  • Array
  • String
  • Wrapper Classes (e.g., Integer, Boolean)
Employee emp = new Employee();  // emp stores reference to object in heap

üß† Interview Tip: In Java, everything is pass-by-value, but with reference types, the "value" passed is the reference to the object, not the object itself.

üî∑ 2. Strings in Java

  • Immutable ‚Üí Once created, cannot be changed.
  • Stored in String Constant Pool (SCP) if declared without new.
  • If declared using new, it‚Äôs stored separately in the heap.
String s1 = "hello";   // goes to SCP
String s2 = "hello";   // refers to same SCP object
String s3 = new String("hello"); // new object in heap

üß† Interview Ask:

  • Difference between == and .equals() on strings?
    • == compares references
    • .equals() compares values

üî∑ 3. Interfaces and Reference Variables

  • You can store child class objects in parent class/interface references.
interface Person { void work(); }

class Engineer implements Person {
    public void work() { System.out.println("Coding..."); }
}

Person p = new Engineer();  // valid
Engineer e = new Engineer(); // valid
// Person p = new Person(); // ‚ùå INVALID (interface can't be instantiated)

üß† Benefit: This enables polymorphism and decoupling (used in frameworks like Spring).

üî∑ 4. Arrays

  • Arrays are also reference types stored in heap.
  • Can store primitive or non-primitive types.
  • Fixed-size and index-based.
int[] arr = new int[5];  // array of size 5
arr[0] = 10;
arr[1] = 6;

üß† Multi-dimensional arrays like int[][] matrix = new int[2][3]; are also supported.

üî∑ 7. Constant Variables

  • Declared using final keyword.
  • Can be static final if value is constant across all objects.

static final double PI = 3.14159;

üß† Used in defining constants, configuration parameters, and as immutable flags.
